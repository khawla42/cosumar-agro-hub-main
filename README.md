```bash
# Frontend
npm install
npm run dev
```

## 🛡️ Sécurité

Le projet inclut des protections contre:
- **Brute Force**: Blocage IP après 5 tentatives échouées
- **SQL Injection**: Détection et blocage des requêtes malveillantes
- **XSS**: Filtrage des scripts malveillants
- **Broken Access Control**: Contrôle d'accès par rôle (Admin, Employé, Client)
- **Parameter Tampering**: Vérification des paramètres pour empêcher les accès non autorisés
