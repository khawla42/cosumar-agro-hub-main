## 🚀 Installation

```bash
# Backend
cd backend
npm install
npm start

# Frontend
npm install
npm run dev
```

## 🛡️ Sécurité

Le projet inclut des protections contre:
- Brute Force (avec blocage IP)
- SQL Injection
- XSS (Cross-Site Scripting)
- Broken Access Control (contrôle d'accès par rôle)
- Parameter Tampering
